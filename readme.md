# AI Chat Agent

A simple command-line chat assistant powered by Groq and LangChain. It can chat and add two numbers using a built-in calculator tool.

## Setup

```bash
pip install -r requirements.txt
```

Create a `.env` file with your Groq API key:

```env
GROQ_API_KEY=your_api_key
```

## Run

```bash
python main.py
```

Type `quit` to exit.
